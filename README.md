# Employee Management App

## Packages and versions
 - `"php": "^8.1"`
 - `laravel/framework": "^10.0"`

## How to run

```
$ git clone https://github.com/s0hel-rana/assesment_test.git
$ cd assesment_test
$ composer update
create database named assesment_test
$ cp .env.example .env
$ php artisan key:generate
$ php artisan migrate
$ php artisan serve

```
